
# ComplaintDesk.AI

ComplaintDesk.AI is a Flutter-based mobile application that allows users to register complaints, upload evidence, track their status, and manage their profile in real-time. The app communicates with a backend server built with Node.js and MySQL.

## Features

- **User Registration & Login**: Secure authentication using email and password.
- **Complaint Registration**: Users can select a category, write a description, and upload evidence.
- **Real-Time Complaint History**: View submitted complaints with status, priority, and timestamps.
- **Profile Management**: Update user information directly in the app.
- **Navigation & UI**: Intuitive navigation between Home, Complaints, Track, and Profile screens.
- **Future Enhancements**: Admin portal notification and advanced feedback system.

## Getting Started

This project is a starting point for a Flutter application. To run it locally, follow these steps:

1. **Clone the repository**
   git clone https://github.com/NawalSajid/complaint_desk_ai.git
   cd complaint_desk_ai

2. **Install dependencies**
   flutter pub get

3. **Run the app**

   * Connect your mobile device via USB or use an emulator.
   * Ensure the backend server is running and your device can access the backend IP.
     
   flutter run

## Backend

* Built with **Node.js**, **Express**, and **MySQL**.
* Handles user authentication, complaint submission, and retrieval.
* Must be running locally or on a server for the app to function properly.
